## 主要竞赛

1. [ACM/ICPC](acm_icpc.md) - 国际大学生程序设计竞赛
2. [挑战杯](challenge_cup.md) - 全国大学生课外学术科技作品竞赛
3. [CTF](ctf.md) - Capture The Flag网络安全竞赛

## 竞赛准备建议

- 根据自己的兴趣和专长选择适合的竞赛方向
- 组建团队，互相学习和鼓励
- 利用学校资源，寻求老师指导
- 参加校内选拔和训练
