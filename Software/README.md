## 🔍 软件选择建议

- **开源优先**：尽可能选择开源软件，避免盗版
- **学生认证**：许多商业软件提供学生免费版，可通过学校邮箱认证
- **云端替代**：考虑使用在线IDE、云数据库等替代本地安装
- **配置统一**：参考推荐配置，与课程要求保持一致
